## Why React Hot Toast?

[React Hot Toast](https://react-hot-toast.com/) is a React library that creates customizable push notifications (called toasts) in our project. You will typically see these notifications when submitting a form, or when given an error message.

React Hot Toast allows us to customize the notification just like any other React component. The library also provides a simple API making them easy to implement.

## Toast Implementation

The `Toaster` component renders all of the toast notifications on the screen. Since we want these notifications to be available on all pages of our website, we put the Toaster component in the layout of our website.

```js title="src/app/layout.js"
import { Toaster } from "react-hot-toast";

export default function RootLayout({ children, session }) {
  return (
    <html lang="en" className="h-full">
      <body className={`${poppins.variable} flex flex-col lg:flex-row h-full`}>
        <Session session={session} refetchInterval={5 * 60} className="h-full">
          <div className="flex w-full">
            <Toaster />
            {children}
          </div>
        </Session>
      </body>
    </html>
  );
}

export default toaster;

```

We created a separate utility file to handle all toast notifications. This utility file contains a `toaster` function that will display a toast notification when called.

The React Hot Toast library allows us to customize our toast notifications using `toast.custom()`. Inside the `.custom()` function, we can customize the toast notifications the same way we create any other React component.

```js title="src/utils/toaster.js"
import { toast } from "react-hot-toast";

const toaster = (message, type) => {
  return toast.custom(
    <div className="flex items-center justify-center bg-white pl-2 pr-5 md:pr-6 py-4 w-4/12 md:w-2/12 shadow-md rounded">
      {icons[type]}
      <div className="inline ps-4 text-sm font-normal text-black">
        {message}
      </div>
    </div>
  );
};

export default toaster;
```

We can import this utility function elsewhere in the project. In the file, when the user hits submit it will make a POST request. Toaster will then display either `Submitted successfully!` or `Internal Server Error` if the POST request is valid.

```jsx title="src/components/dynamic/form/Admin.jsx"
import toaster from "@/utils/toaster.js";
const handleSubmit = (setLoading, setState) => {
  api({
    method: "POST",
    url: "/api/dashboard/admins",
    body: admin,
  })
    .then(() => toaster(`Submitted successfully!`, "success"))
    .catch(() => toaster(`Internal Server Error`, "error"))
    .finally(() => {
      setLoading(false);
      setState(2);
    });
};
```
